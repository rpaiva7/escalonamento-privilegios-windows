## Escalonamento de Privilégios no Windows

![hacking](https://www.sandsteinwandern.de/wandern/wp-content/uploads/2019/11/giphy-hack2.gif)

O escalonamento de privilégios é uma das técnicas mais comuns que os invasores usam para descobrir e exfiltrar dados valiosos e confidenciais. Nesta técnica o invasor geralmente entra como usuário, mas vai adquirindo permissões de administrador sem a necessidade de autenticação e comprovação dessas permissões.

Para um hacker, o escalonamento de privilégios é a arte de aumentar os privilégios desde o acesso inicial, normalmente um usuário padrão ou conta de aplicativo, até o administrador, root ou até conseguir acesso total ao sistema.
